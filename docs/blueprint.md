# **App Name**: Result Ruse

## Core Features:

- Login Page Visual Replication: Create a UI that closely mimics the official CBSE portal login page, including the specified header text ('केन्द्रीय माध्यमिक शिक्षा बोर्ड', 'Central Board of Secondary Education'), subtitle ('Examination Results — Brought to you by National Informatics Centre'), and the emblem's positioning.
- Dynamic User Input Form: Implement form fields for Roll Number, School No., Date of Birth (dd/mm/yyyy), Admit Card ID, and Candidate Name, with styling consistent with the provided reference image.
- Form Submission Handling: Capture user-entered data from the login form upon clicking the 'Submit' button.
- Result Page Display: Dynamically display the submitted user's Roll Number, School No., Date of Birth, Admit Card ID, and Candidate Name on a dedicated results page.
- Prank Result Content: Present a pre-determined, static, fake result or status message on the results page to create the prank effect.
- Page Navigation: Manage the transition between the login and results pages upon form submission.

## Style Guidelines:

- A light color scheme to emulate official academic portals. The header features a vibrant teal/cyan as per the request, chosen to reflect trust and clarity (Primary color: #2EC1C1).
- A soft, barely-there tint of teal for the background, creating a clean and spacious canvas while subtly echoing the header (Background color: #ECF6F6).
- A contrasting yet harmonious dark blue serves as an accent, providing visual weight for interactive elements and key information (Accent color: #146CCF).
- Body and headline font: 'Inter' (sans-serif) for its modern, neutral, and highly legible appearance, suitable for official documents and a clean user experience.
- Include a placeholder for the CBSE emblem to match the visual presentation of the official portal. This emblem image will need to be provided by the user.
- Mimic the central alignment and structured form layout of the reference image for the login page, transitioning to a similar clean layout for the result display.
